# ICU Readmission Prediction (eICU-CRD Demo)

This repository contains the implementation and project artefacts for an MSc dissertation on **machine learning-based readmission risk prediction** using the **eICU Collaborative Research Database Demo (eICU-CRD Demo v2.0.1)**.

The work builds an end-to-end, reproducible pipeline covering:
- dataset preparation and preprocessing
- feature engineering
- training and comparing multiple ML models
- evaluation with imbalance-aware metrics (ROC-AUC, PR-AUC, MCC, etc.)
- interpretability (feature importance / SHAP/LIME-style explanations)
- dissertation-ready outputs (plots/tables) and project management evidence

---
